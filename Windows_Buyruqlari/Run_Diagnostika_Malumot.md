🛠 8. Diagnostika va Tizim Axborot Buyruqlari
| Buyruq                        | Nima qiladi                                                                  |
| ----------------------------- | ---------------------------------------------------------------------------- |
| dxdiag                        | DirectX diagnostika vositasi (grafika, ovoz va tizim haqida ma’lumot beradi) |
| msinfo32                      | Tizim haqida keng ma’lumot (CPU, RAM, BIOS, drayverlar va h.k.)              |
| perfmon /report               | Kompyuterning ishlash hisobotini yaratadi (60 soniyada)                      |
| eventvwr                      | Tizimdagi voqealarni (xatolik, ogohlantirish) ko‘rsatadi                     |
| powercfg /batteryreport       | Noutbuk batareya holati haqida hisobot yaratadi                              |
| powercfg /energy              | Quvvat samaradorligi haqida hisobot                                          |
| tasklist                      | Kompyuterda ishlayotgan barcha dasturlar va jarayonlar ro‘yxati              |
| taskkill /f /im notepad.exe   | Belgilangan ilovani (masalan, Notepad) majburan yopadi                       |
| verifier                      | Drayverlar monitoringi (faqat tajribali foydalanuvchilar uchun)              |
| whoami                        | Hozirgi foydalanuvchi nomini ko‘rsatadi                                      |
