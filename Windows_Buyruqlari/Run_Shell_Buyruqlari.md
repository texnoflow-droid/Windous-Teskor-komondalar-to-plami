🧠 1. shell: Buyruqlari — Ilovalar va papkalarni ochish
| Buyruq                      | Vazifasi                                                 |
| --------------------------- | -------------------------------------------------------- |
| shell:appsfolder            | Barcha ilova va dasturlar ro‘yxatini ko‘rsatadi          |
| shell:startup               | Windows boshlanishida ishga tushadigan dasturlar papkasi |
| shell:common startup        | Hamma foydalanuvchilar uchun avto-ishga tushadiganlar    |
| shell:sendto                | “Yuborish (Send to)” menyusini tahrirlash papkasi        |
| shell:recent                | Yaqinda ochilgan fayllar ro‘yxati                        |
| shell:downloads             | Yuklab olingan fayllar papkasi                           |
| shell:programs              | Start menyusidagi dasturlar                              |
| shell:desktop               | Ish stoli (Desktop) papkasi                              |
| shell:common desktop        | Hamma foydalanuvchilar ish stoliga fayllar qo‘yish       |
| shell:mycomputerfolder      | Bu "This PC" oynasini ochadi                             |
| shell:windows               | Windows o‘rnatilgan papkani ochadi (odatda C:\Windows)   |
| shell:profile               | Sizning foydalanuvchi papkangiz                          |
| shell:common programs       | Barcha foydalanuvchilarning Start menyusi                |
| shell:history               | Internet Explorer tarixini ochadi                        |
| shell:fonts                 | Shriftlar (Fonts) oynasi                                 |
| shell:controlpanel          | Klassik boshqaruv paneli                                 |
| shell:NetworkPlacesFolder   | Tarmoq manzillari oynasi                                 |
