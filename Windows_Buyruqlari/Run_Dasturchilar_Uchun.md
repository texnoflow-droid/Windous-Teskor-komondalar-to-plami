🧰 4. Dasturchilar uchun Run buyruqlari
| Buyruq                         | Nima qiladi                                           |
| ------------------------------ | ----------------------------------------------------- |
| notepad                        | Notepad matn muharriri                                |
| write                          | WordPad ilovasi                                       |
| code                           | Visual Studio Code (agar o‘rnatilgan bo‘lsa)          |
| explorer .                     | Joriy papkani File Explorer’da ochadi                 |
| powershell                     | PowerShell terminali                                  |
| cmd /k                         | Buyruq qatori oynasini ochadi va buyruq kutadi        |
| tasklist                       | Yurgan jarayonlar ro‘yxatini ko‘rsatadi               |
| taskkill /im [dastur.exe] /f   | Belgilangan dastur jarayonini majburan yopadi         |
| ping [sayt.com]                | Saytga signal yuborish orqali bog‘lanishni tekshiradi |
| ipconfig                       | IP manzilingiz va tarmoq ma’lumotlarini ko‘rsatadi    |
| netstat -an                    | Ochiq portlar va ulanishlarni ko‘rsatadi              |
